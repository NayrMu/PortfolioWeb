Command Line Interface File System Manager

This is a Command Line Interface (CLI) application that allows you to navigate through your file system similar to powershell on windows or bash on Linux. It uses C# and the .NET Framework with Object Oriented Programming (OOP) concepts to create a simple yet versatile CLI tool. The syntax is a little different than standard command line and but it was a great project for learning user input sanitization and parsing. I have the code posted on github and documentation below.
